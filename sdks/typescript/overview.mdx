## Overview

### Available Operations

* [postMessage](#postmessage) - Post a new message and publish to a message queue
* [getMessages](#getmessages) - Retrieve a list of messages
* [getMessageById](#getmessagebyid) - Retrieve details of a specific message
* [queueMessage](#queuemessage) - Queue a message for sending at a specified time
* [getSendStatus](#getsendstatus) - Get the status of a queued message send operation
* [postSubscriber](#postsubscriber) - Create a subscriber and handle their subscription status
* [getSubscriber](#getsubscriber) - Retrieve a subscriber's details
* [updateSubscriber](#updatesubscriber) - Update subscriber details
* [deleteSubscriber](#deletesubscriber) - Delete a subscriber
