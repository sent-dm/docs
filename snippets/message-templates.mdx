### `id` (string, required)
The unique identifier for the message template, automatically generated and managed.

### `messageId` (string, required)
The unique identifier of the message to which the template belongs.

### `channelType` (string, required)
The type of channel for the template.
- Values: 'sms', 'whatsapp', 'telegram', 'imessage'

### `content` (object, required)
The content of the template. Can include various properties depending on the message.

### `status` (string, required)
The status of the template.

### `createdAt` (DateTime, required)
The timestamp indicating when the template was first created in the system, formatted as a date-time string.

### `updatedAt` (DateTime, required)
The timestamp indicating when the template information was last updated, formatted as a date-time string.

### `name` (string, required)
The name of the template.
